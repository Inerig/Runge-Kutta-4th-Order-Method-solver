Instructions:

  • Use t for the independent variable (time).
  • Use y for the dependent variable.
  • Use Math for functions like Math.exp, Math.log, Math.sin.
  • Avoid spaces between operators and variables.

Examples:

  • Linear: t - y
  • Exponential: Math.exp(t) - y
  • Logarithmic: Math.log(y)
  • Trigonometric: Math.sin(t) + Math.cos(y)
  • Hyperbolic: Math.sinh(t) - Math.cosh(y)
  • Power: Math.pow(t, 2) + Math.pow(y, 3)
  • Rational: t / y
  • Piecewise: (t < 0) ? t : y
